## **About**:
#### This repository contains two folders for front-end and back-end development of a mini-ebay like project, ***brought to you by T-17***.


## **Getting Started:**
1. Clone Repository
2. In Backend folder:
 * npm install
 * npm start
3. In Frontend folder:
 * npm install
 * npm start or ng serve (according to your config)
